八年后，LeftGeek的（短暂）回归

分类: 杂项
标签:  Java 网站建设
作者
余亚
　　距离上一次12年圣诞节建站时发的第一篇博文，已经过去了8年多了。

　　期间网站关闭，除了学业繁忙的原因，主要还是因为负担不起服务器。当时是用学校实验室的服务器来运行网站，可后来学校规定（应该是响应国家的网站备案政策，无可厚非）使用校内的服务器做网站必须要去登记，显然这种拿占用公共资源的私人行为不会被批准，于是自那以后就停办了。
　　而关于域名的更换，本来一个域名一年的费用也就百把块钱，对于穷学生来讲也算可以接受。但是因为域名的续费要比首次购买贵一些，所以在14年就换了域名，从当时的lefthacker.com换成了现在的leftgeek.com，意思一样但是更为简洁。

　　之所以现在重新上线网站，其实也是机缘巧合。
　　最近即将毕业，在为找工作准备之际，整理了自己以前做过的一些项目。 其中一个项目就是当年备份了的JAVA博客网站，抱着试一试的心态，快速在本地搭建了JDK + Tomcat服务器 + MySQL数据库的运行环境。
　　在还原了数据库内容后，网站居然运行成功了。要知道运行环境的各种库版本早已与当年相差很多，看来它们的兼容性做得不错。
　　然后就看到了当年发表的第一篇博文，感慨当时自己对学习新技术的激情。虽然充满中二装*的气息，但是贵在真实。
　　于是老夫聊发少年狂，一激动就重新买了域名和云服务器。这里得插入一句：本来虚拟主机就够用了，毕竟比云服务器便宜些，奈何它们都只支持LAMP（Linux + Apache + MySQL + PHP）的网站架构，而不是本网站所用的JAVA。
　　最后在成功备案后，leftgeek.com网站就又上线了。不过由于云服务器太贵了，目前只买了几个月的，后面有稳定收入了再考虑续。
 
　　回想过去读博的这8年，就像经历了电影《太空旅客》，本来乘着飞船要去光年之外的外星球，结果中途因休眠舱故障提前醒来。
 
　　到不了的终点，回不去的起点，只能就地重新开始。
 
　　这就回到本网站的定位上面来，很遗憾这个问题目前仍然没有答案。移动互联网时代，大家获取信息的方式早就从桌面端转向了移动端。这个网站也已是明日黄花：它并没有为移动端的阅读做优化，而且厚重阴影的立体风格也与当下主流的扁平化审美格格不入。
 
　　当然这个倒是其次的，因为不论形式如何改变，永远是内容为王。可惜做了这么多年学生，没能在某个领域深耕，也暂时无法保证高质量的内容产出。不过考虑到我并没想过未来靠它赚钱，因此对于发表的内容可以有最大的自由度（只要脸皮够厚）。就目前而言，该网站仍然只是自嗨的产物。
　　但是话说回来，这么多年还是多多少少积累了一些零零碎碎的内容，有些日后整理好了还是可以分享的。除此以外，这个网站的源码，以及其它的一些放到github上的小项目，都可以聊聊其中有意思的细节。毕竟开源项目远不止是开放源代码这么简单，没有文档，没有更新那就只是自嗨罢了。
 
　　但是，还是请别抱有任何期待。最近仍然在为找工作做准备，并没有太多时间更新网站内容。至于未来，如果没有被996压垮，你应该还能见到我。
